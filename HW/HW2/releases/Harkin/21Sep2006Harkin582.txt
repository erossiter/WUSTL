   The tribal members   contribution to communication during the two world wars was invaluable in relaying information as the enemy was never able to break the code  The use of these codes is credited with saving countless lives   Eight members of the Meskwaki tribe from Tama County served as code talkers for eight companies of their division in North Africa during WWII         Until 1968  information related to code talkers activities remained classified   This postponement caused significant delay in learning more about the role of code talkers and delayed appropriate recognition  In 2000  a law was enacted to present congressional medals to the Navajo code talkers  Since then  more information has come forward confirming that some 17 additional tribes participated in code talking  including the Meskwaki                  Originating Site  Number  Site Filter  Search
